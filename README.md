# 43-F: SWIR/NIR Spectral Micro-Beads for Autonomous Search and Rescue Detection

> **Part of [Project 43 — Labrador](https://github.com/aaronukgarcia): Exploring commercial and scientific applications of synthetic labradorescence**

## Overview

Proposes rigid glass or polymer micro-beads coated with a low-&Delta;n multilayer thin-film structure tuned to SWIR/NIR wavelengths (900–1700 nm) for drone-based SAR detection. Critical caveat: preliminary estimates suggest geometric scattering loss from spherical geometry likely exceeds spectral gain.

## Key Idea

Narrow-band spectral micro-beads could provide a spectrally selective SAR marker detectable by drone-mounted InGaAs cameras, filling a gap in current SOLAS/IMO standards which specify only visible-spectrum retroreflection. The spherical geometry is the simplest starting point but may need to pivot to platelet or retroreflector geometries.

## Target Applications (Conditional)

- Maritime SAR
- Wilderness SAR
- Military personnel recovery
